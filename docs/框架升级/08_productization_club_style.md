# 产品化结构对标（俱乐部广告可学点）

> 目标：学“结构化交付与节奏”，不学“杠杆叙事与不可复核收益目标”。

## 可学点（结构层）

1) **候选池 → 组合 → 执行 → 复盘** 的闭环结构  
2) **风控钩子前置**（先控风险再谈收益）  
3) **信息分层**：数据层（信号）/ 规则层（组合与风控）/ 解读层（解释与注意事项）  
4) **固定节奏交付**（每日固定时间出简报）

## 不学点（风险层）

- “5天持股+3%目标+仓位3%~130%”的高杠杆叙事  
- 直接用凯利公式生成实盘仓位  
- AI叙事替代可验证数据

## 落地清单（已完成/待扩展）

### ✅ 已落地：每日简报（结构化交付）

- 新增命令：`llm_trading daily-brief`  
- 产出：`outputs/agents/daily_brief.md`  
- 内容层级：
  - 候选池 TopN（ETF / 个股）
  - 模拟持仓（<=3，只做规则版）
  - 执行草案（orders_next_open）
  - 风控/警告（warnings）

### ⏳ 待扩展（可选）

1) **固定日程驱动**  
   - 在 `run` 结束后自动触发 `daily-brief`（可加 `--with-daily-brief`）  
2) **资金/舆情附录**  
   - 选定标的补齐资金面/舆情摘要（非强制，引用公告优先）  
3) **周度复盘**  
   - 从 `monitor` 汇总周报（胜率/回撤/滑点/执行偏差）
