# 中低频量化策略资源（偏趋势/动量/TAA）

目标：给本仓库的“收盘后跑脚本、次日手工决策”的使用方式，找一堆**成熟的参考坐标系**（思路、指标、组合方式、回测口径）。

## 国外（偏研究/策略库/组合跟踪）
- **Quantpedia**：策略百科 + 文献证据链（适合找“某策略是否真有统计优势”与实现要点）。
- **AllocateSmartly**：偏 TAA/轮动/动量的模型组合跟踪（适合对标“如何把策略变成可执行组合”）。
- **Quantocracy**：量化内容聚合（适合扫热点，但需要自己甄别质量）。
- **AQR Research**：趋势/动量等研究文章（适合拿来做“策略为什么有效”的理论支撑）。
- **Alpha Architect**：因子/趋势/资产配置相关研究与博客（实现细节比较落地）。
- **Meb Faber**：趋势跟随/配置类文章（很适合做“右侧定方向”的框架参考）。
- **Dual Momentum / OptimalMomentum**：Dual Momentum 体系的落地资料（适合参考“风险开关 + 轮动”的结构）。

## 国内（偏社区/回测平台）
- **聚宽 JoinQuant / 米筐 RiceQuant / 优矿 Uqer / 掘金 QuantX / BigQuant**：A 股回测与社区分享多，适合搜“ETF 轮动/趋势/动量/熊市过滤”的现成实现与坑点。

## 建议阅读顺序（最省时间）
1) 先看 Meb/AQR 把“右侧趋势”框架吃透 → 2) 再看 Dual Momentum/AllocateSmartly 学“风险开关 + 组合执行” → 3) 最后用 Quantpedia/国内社区补“证据链 + 参数坑”。
